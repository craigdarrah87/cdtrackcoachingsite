# CD Track Coaching — Next.js + Tailwind

Raw, enthusiast-built website for Nürburgring & European track-day coaching.

## Quick start

```bash
npm i
npm run dev
```

## Deploy to Vercel
1. Push this folder to a GitHub repo.
2. Import the repo in Vercel.
3. Framework preset: **Next.js** (App Router).
4. Build command: `next build` (default).

## Content & Assets
- Primary colours: **black / white / red (#ff2a2a)**
- Local racing fonts located in `public/fonts/`
- Images in `public/images/`
